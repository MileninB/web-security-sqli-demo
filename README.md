# К.Т.4 - SQL Injection Demo

Учебный проект по дисциплине «Безопасность веб-приложений».

## Реализовано
- уязвимая версия поиска пользователей;
- демонстрация SQL-инъекции;
- демонстрация UNION-атаки;
- защищённая версия с параметризованными запросами;
- валидация входных данных.

## Запуск уязвимой версии
node app-vuln.js

## Запуск защищённой версии
node app-secure.js
